# 🎓 Smart Campus AI

Smart Campus AI is a Streamlit-based web application that helps students manage their campus activities using AI.

## ✨ Features

- 🔐 User Login & Registration
- 🤖 AI Chatbot
- 📅 Timetable Management
- 📢 Announcements
- 📝 Assignments
- 📊 Attendance Tracking
- 👤 Student Profile
- ⚙️ Settings

## 🛠 Technologies Used

- Python
- Streamlit
- JSON Database
- Git
- GitHub

## ▶️ Installation

1. Clone the repository

```bash
git clone https://github.com/akshithasalvoju/SmartCampusAI.git
```

2. Open the project

```bash
cd SmartCampusAI
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
streamlit run app.py
```
